# LabHub 🧪💻

O **LabHub** é um sistema desenvolvido em Django para o gerenciamento e agendamento de laboratórios. O objetivo principal é facilitar a reserva de espaços, controle de equipamentos e organização de horários de forma simples e intuitiva.

## 🚀 Funcionalidades Planejadas
- [x] Estrutura básica do projeto Django configurada
- [ ] Cadastro e autenticação de usuários (professores, alunos e administradores)
- [ ] Sistema de agendamento de horários para laboratórios
- [ ] Gerenciamento de equipamentos disponíveis

## 🛠️ Tecnologias Utilizadas
- **Python**
- **Django**
- **HTML5 / CSS3**
